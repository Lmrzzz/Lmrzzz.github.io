斗地主手牌排序
困惑：
  如何将一个字符串列表(扑克牌)进行排序呢？
思路：
  使用sort 重写sort方法(利用lambda进行简化)
遇到的困难：
  1没有将字母转化为数字的思路，无法进行排序;2大小王该如何去排序呢？
收获:
  1熟练调用String的Integer.parseInt api
  2重写sort的思路
  3熟悉lambda表达式基本简化

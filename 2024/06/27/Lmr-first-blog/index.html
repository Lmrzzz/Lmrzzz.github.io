Java基础篇–初认sort 斗地主手牌排序

title: chenge-first-blog
date: 2021-07-22 14:08:14
tags:

困惑：
如何将一个字符串列表(扑克牌)进行排序呢？
思路：
使用sort 重写sort方法(利用lambda进行简化)
遇到的困难：
1没有将字母转化为数字的思路，无法进行排序;2大小王该如何去排序呢？
收获:
1熟练调用String的Integer.parseInt api
2重写sort的思路
3熟悉lambda表达式基本简化

关键代码:
static public ArrayList unJokerSort(ArrayList arrayList){
arrayList.sort((o1, o2) -> {
String s1=o1.substring(1);
String s2=o2.substring(1);
int x1=conVert(s1);
int x2=conVert(s2);
return x1-x2;
});
return arrayList;
}
其中convert函数:
static public int conVert(String s){
switch (s){
case “10”->{
return 10;
}
case “J”->{
return 11;
}
case “Q”->{
return 12;
}
case “K”->{
return 13;
}
case “A”->{
return 14;
}
case “2”->{
return 15;
}
default -> {
for (char c:s.toCharArray()){
if (Character.isDigit(c)){
int num=Integer.parseInt(s);
return num;
}
}
}
}
return 0;
}
